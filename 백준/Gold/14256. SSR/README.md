### 문제 설명

<p>SSR(A, B) = (sqrt(A) + sqrt(B))^2 이다.</p>

<p>두 정수 N, M이 주어졌을 때, 1 ≤ A ≤ N, 1 ≤ B ≤ M 이면서 SSR(A, B)가 정수인 (A, B) 쌍의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 N, M이 주어진다. (1 ≤ N, M ≤ 77,777)</p>

### 출력 

 <p>첫째 줄에 문제의 정답을 출력한다.</p>
